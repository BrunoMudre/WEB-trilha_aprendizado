  • table

  • ins
  • sub
  • sup
  • blockquote
  • q
  • abbr
  • address

< table >

Permite criar tabelas em uma página HTML, estruturando os dados em linhas e colunas.

A tabela é criada a partir da tag <table> e, para finalizar, utiliza-se a tag </table> ;

Para delimitar a área do cabeçalho da tabela, utiliza-se o elemento <thead> ;

Para delimitar o corpo/área principal da tabela, utiliza-se a tag <tbody> ;

A tag <tr> delimita as linhas da tabela;

A tag <td> insere os dados e separa nas colunas desejadas;

A tag <th> insere o cabeçalho na tabela, dando destaque e identificando cada coluna específica;

Para adicionar uma legenda ou título à tabela, usa-se a tag <caption>;

E o elemento <tfoot> delimita o rodapé da tabela;

Para a mesclagem de colunas, usa-se o atributo colspan, e, para linhas, rowspan;

O atributo border insere a borda para identificar a separação das linhas e colunas.

<del> e <ins>

<del> Define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.

<ins> Define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Exemplo:

Minha cor favorita é vermelho azul.

Sintaxe:

<sub>

Define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas.

Exemplos:

H2O

Texto subscrito

Sintaxe:

<sup>

Define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

Exemplo:

Texto sobrescrito

x2

Sintaxe:

<blockquote>

Especifica uma seção que é citada de outra fonte.

Exemplo:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

Sintaxe:

<q>

Assim como o blockquote, define uma citação, porém desta vez uma citação curta.

Exemplo:

O objetivo deles é: Construir um futuro onde as pessoas vivam em harmonia com a natureza. Esperamos que eles tenham sucesso.

Sintaxe:

<abbr>

Define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Exemplo:

A OMS foi fundada em 1948.

Sintaxe:

<address>

Define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.

Exemplo:

Escrito por Jon Doe.
Nos visite em:
Exemplo.com
Box 564, Disneyland
USA

Sintaxe:

<cite>

Define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

Exemplo:

O Grito de Edward Munch. Pintado em 1893.

Sintaxe:

<bdo>

É usada para substituir a direção do texto atual.

Exemplo:

Esse texto vai da direita para esquerda.

Esse texto vai da esquerda para direita.

Sintaxe:

<code>

É usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Obs: Esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS.

Exemplo:

A tag button do HTML define um botão clicável.

A propriedade background-color do CSS define a cor de fundo de um elemento.

Sintaxe:

<pre>

O texto em um <pre> elemento é exibido em uma fonte de largura fixa e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.

Exemplo:

O texto em um <pre> elemento
        é exibido em uma  fonte de largura fixa
        e o texto
        preserva      espaços e 
        quebras de linha.

Sintaxe:

<kbd>

É usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Obs.: Esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS.

Exemplo:

Pressione Ctrl + C para copiar texto (Windows).

Pressione Cmd + C para copiar texto (Mac OS).
